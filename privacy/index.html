Privacy Policy

Effective Date: August 18, 2025  |  Last Updated: August 18, 2025

Overview

YourTake is a Chrome extension providing personalized, goal-focused insights from YouTube videos and articles. This policy explains how the service collects, uses, and protects user information.

Information We Collect

1. Profile Information

(Server Stored) (Stored Locally)

  • User Goals: Personal and professional objectives
  • Role/Position: Current role or field of work
  • Areas of Interest: Topics and subjects for focus
  • Projects: Current projects in progress

Storage: Information stored locally on device via Chrome's storage API. Data is NOT transmitted to servers.

2. Usage Analytics

Anonymous usage data collected for service improvement:

  • Extension Events: Installation, side panel usage, content analysis actions
  • Content Interaction: Feature usage (copy insights, expand sections, rate content)
  • Performance Data: Analysis success rates and error tracking
  • Anonymous Identifiers: Randomly generated IDs to track usage without identifying users

Third Party: Analytics data sent to Google Analytics 4 using their Measurement Protocol.

3. Content Access

  • Web Page Content: Publicly available content from articles and YouTube videos selected for analysis
  • Tab Information: Current tab URL and title for analysis context

Usage: Content processed locally; sent to analysis service only when explicitly requested.

How We Use Your Information

Profile Information

  • Generate personalized insights based on goals and interests
  • Filter content relevance to specific needs
  • Improve analysis accuracy over time

Usage Analytics

  • Understand user interaction patterns
  • Identify and fix technical issues
  • Improve feature performance and user experience
  • Track new feature adoption

Content Data

  • Extract relevant insights from videos and articles
  • Generate author highlights and key takeaways
  • Provide personalized recommendations

Data Sharing and Third Parties

Google Analytics 4

Anonymous usage data shared for:

  • Data Shared: Anonymous usage events, feature interactions, error tracking
  • Purpose: Service improvement and performance monitoring
  • User Identification: No personally identifiable information shared
  • Google's Policy: Subject to Google's privacy practices

Content Analysis Service

When requesting content analysis:

  • Data Shared: Article text or YouTube transcript, profile preferences
  • Purpose: Generate personalized insights and recommendations
  • Processing: Content processed but not permanently stored
  • Security: Data transmission encrypted via HTTPS

No Sale of Data

The service does NOT sell, rent, or trade personal information to third parties for commercial purposes.

Data Security

Local Storage Protection

  • Profile data stored using Chrome's secure storage APIs
  • Data encrypted and isolated per browser profile
  • No transmission to external servers without explicit user action

Transmission Security

  • All external communications use HTTPS encryption
  • Anonymous identifiers prevent personal identification
  • Minimal data transmission principle applied

Data Retention

Profile Information

  • Stored locally until extension removal or browser data clearing
  • Under complete user control via Chrome's extension management

Analytics Data

  • Anonymous usage data subject to Google Analytics retention policies
  • Typically retained for 26 months per Google's terms

Content Analysis

  • Processed content not permanently stored
  • Analysis results cached locally for improved performance

Your Rights and Controls

Access and Control

  • View Profile: Access all stored profile information via extension options
  • Edit Profile: Modify goals, interests, and preferences at any time
  • Delete Data: Remove extension to delete all locally stored data

Analytics Opt-Out

Currently, usage analytics support service improvement. Future versions may include opt-out options.

Content Analysis Control

  • Explicit Consent: Analysis occurs only when clicking "Analyze" buttons
  • Content Choice: User controls which content gets analyzed
  • Result Management: Delete analysis results anytime by re-analyzing content

Children's Privacy

YourTake is not directed at children under 13. The service does not knowingly collect personal information from children under 13. If awareness of such collection occurs, that information will be deleted.
